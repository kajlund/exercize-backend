# Sportz - Backend

REST API server for the sportz app implemented using Node.js/express, Drizzle ORM amd Postgres SQL DB.

### DBMS Commands

- Updating DB - `npx drizzle-kit push`
- Browsing DB - `npx drizzle-kit studio`
- Generate migrations: `npx drizzle-kit generate`
- Apply migrations: `npx drizzle-kit migrate`
